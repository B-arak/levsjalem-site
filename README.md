# שלם מלב שבור - אתר הספר

אתר רשמי למדריך "שלם מלב שבור" – מדריך רגשי לגרושים ופרודים טריים.

בנוי באמצעות Next.js, TypeScript ו‑Tailwind CSS. המטרה היא לספק חוויית קריאה מודרנית, רספונסיבית וידידותית לקידום ברשת.

## התחלת עבודה

הריץ את שרת הפיתוח:

```bash
npm install    # התקנת תלויות בפעם הראשונה
npm run dev
```

פתח את [http://localhost:3000](http://localhost:3000) בדפדפן.

את התוכן הראשי ניתן לערוך ב-`src/app/page.tsx`, בעוד שהפריסה ואמצעי הניווט נמצאים ב־`src/app/layout.tsx`.

### נכסים חזותיים וקבצים

- העלה את תמונת כיסוי הספר לתיקייה `public` בשם `cover.png`.
- אם יש לך PDF של התקציר או הפרק הראשון, שמור אותו כ-`public/first-chapter.pdf` כדי שהקישור בהורדה יעבוד.

## מבנה הפרויקט

- `src/app/` – תיקיית App Router של Next.js.
- `src/components/` – רכיבים משותפים כגון כותרת תחתונה (Header/Footer).
- `src/app/globals.css` – סגנונות גלובליים (Tailwind + משתני CSS).

## פונטים

הפרויקט כולל שימוש בגוגל פונטים `Montserrat` לכרזות ו־`Roboto` לטקסט גוף.

## פריסה

האתר בנוי במצב RTL (עברית) ומתממשק היטב לנייד ודסקטופ.

## פריסה ושדרוג

קידום ברשת מבוצע על ידי תוכן קריא, תגיות מטא, וערכת קישורים ברורים לרכישה.

## פרסום

ניתן לפרסם ב‑Vercel, Netlify או כל שירות התומך ב‑Next.js.
